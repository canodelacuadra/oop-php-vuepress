# Instanciar objetos mediante función constructora

Los objetos se instancian con la palabra reservada new+<el nombre de la clase>+(). Si les pasamos los parámetros, estaremos invocando a la función contructora y definiremos ese objeto dando valor a esas propiedades

Una función constructora o constructor, es una función que podemos llamar al mismo tiempo que instanciamos un objeto. Un constructor nos permite instanciar un objeto inicializando sus propiedades. Esto nos evitará tener que ir inicializando uno a uno los valores de cada una de las propiedades del objeto instanciado.

Si un objeto tiene una función constructora function __construct(), PHP llamará automáticamente a esta función cuando se instancie un objeto a partir de una clase.

La función construct comienza con dos guiones bajos (__). Esta sintaxis es propia de los métodos mágicos en php (opens new window)

Veamos en el siguiente ejemplo, que el uso de un constructor nos evita llamar al método set_name() una vez instanciado el objeto y reduce por tanto la cantidad de código:

Ejemplo

<?php
class Fruit {
  public $name;
  public $color;

  function __construct($name) {
    $this->name = $name;
  }
  function get_name() {
    return $this->name;
  }
}

$manzana= new Fruit("manzana");
echo $manzana->get_name();
?>

En este ejemplo además del nombre, incorporamos el color al método constructor

Ejemplo

<?php
class Fruit {
  public $name;
  public $color;

  function __construct($name, $color) {
    $this->name = $name;
    $this->color = $color;
  }
  function get_name() {
    return $this->name;
  }
  function get_color() {
    return $this->color;
  }
}

$manzana = new Fruit("Manzana", "red");
echo $manzana->get_name();
echo "<br>";
echo $manzana->get_color();
?>

# La función destructora

Esta función se invoca (si existe) cuando se destruye el objeto o se detiene o sale del script.

Si crea una función __destruct(), PHP llamará automáticamente a esta función al final del script.

La función de destrucción comienza con dos guiones bajos (__).

El siguiente ejemplo tiene una función __construct () que se llama automáticamente cuando crea un objeto de una clase, y una función __destruct () que se llama automáticamente al final del script:

<?php
class Fruit {
  public $name;
  public $color;

  function __construct($name) {
    $this->name = $name;
  }
  function __destruct() {
    echo "Se ha creado la fruta {$this->name}.";
  }
}

$apple = new Fruit("Apple");
?>

En esta función destructora nos confirma también que no solo se ha dado valor al name , sino también un color al color

<?php
class Fruit {
  public $name;
  public $color;

  function __construct($name, $color) {
    $this->name = $name;
    $this->color = $color;
  }
  function __destruct() {
    echo "Esta fruta es {$this->name} y su color es {$this->color}.";
  }
}

$apple = new Fruit("Apple", "red");
?>

Los constructores y destructores ¡son muy útiles!: ayudan a reducir la cantidad de código que debemos escribir.
