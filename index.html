# Introducción a la Programación Orientada a Objetos en PHP - OOP

Cuando comenzamos a escribir un programa pequeño, es posible que solo necesitemos juntar un par de líneas de código php, luego agruparlas en funciones y cuantas más funciones escribimos, podemos sentir la necesidad para organizarlos mejor en clases.

OOP simplemente deja que estructuremos nuestros códigos mejor, lo que permite un mejor mantenimiento del código.

Las clases y los objetos son los dos pilares principales de la programación orientada a objetos.

Una clase es simplemente una forma de almacenar funciones (también conocidas como métodos) y varibles (también conocidas como propiedades) como código PHP en uno o más "fragmentos" relacionados donde cada "fragmento" individual trata un tema específico o pieza de funcionalidad.

Um objeto es el uso específico de una clase (o varias clases), una instancia de la misma.
