# Qué son los nameSpaces

Los espacios de nombres resuelven dos problemas diferentes:

  1. Permiten una mejor organización al agrupar clases que trabajan juntas para realizar una tarea.
  2. Permiten utilizar el mismo nombre para más de una clase.

Por ejemplo, podemos tener un conjunto de clases que describen una tabla HTML, como Tabla, Fila y Celda, mientras que también podemos tener otro conjunto de clases para describir muebles, como Mesa (Table), Silla y Cama.

Los espacios de nombres se pueden usar para organizar las clases en dos grupos diferentes y al mismo tiempo evitar que se mezclen las dos clases Table html y Table mueble.

# Declarar un espacio de nombres

Los espacios de nombres se declaran al principio de un archivo usando la palabra clave namespace:

Declaramos un espacio de nombres llamado Html:

namespace Html;

Nota: una declaración namespace debe ser lo primero en el archivo PHP. El siguiente código no sería válido:

<?php
echo "Hello World!";
namespace Html;
...
?>

Las constantes, clases y funciones declaradas en este archivo pertenecerán al espacio de nombres Html :

Ejemplo Creamos una clase de tabla en el espacio de nombres Html:

<?php
namespace Html;
class Table {
  public $title = "";
  public $numRows = 0;
  public function message() {
    echo "<p>La Table '{$this->title}' tiene {$this->numRows} filas.</p>";
  }
}
$table = new Table();
$table->title = "My table";
$table->numRows = 5;
?>

<!DOCTYPE html>
<html>
<body>

<?php
$table->message();
?>

</body>
</html>

Para una mayor organización, es posible tener espacios de nombres anidados:

Sintaxis Declare un espacio de nombres llamado Html dentro de un espacio de nombres llamado Código:

namespace Code\Html;

# Usar espacios de nombres

Cualquier código que siga a una declaración namespace está operando dentro del espacio de nombres, por lo que las clases que pertenecen al espacio de nombres se pueden instanciar sin ningún calificador. Para acceder a clases desde fuera de un espacio de nombres, la clase debe tener el espacio de nombres adjunto.

Ejemplo Utilice clases del espacio de nombres Html:

$table = new Html\Table()
$row = new Html\Row();

Cuando se utilizan muchas clases del mismo espacio de nombres al mismo tiempo, es más fácil utilizar la palabra clave namespace:

Ejemplo Utilice clases del espacio de nombres Html sin la necesidad del calificador Html :

namespace Html;
$table = new Table();
$row = new Row();

# Alias ​​de espacio de nombres. USE

Puede resultar útil asignar un alias a un espacio de nombres o una clase para facilitar la escritura. Esto se hace con la palabra clave use:

Ejemplo Asigne un alias a un espacio de nombres:

use Html as H;

$table = new H\Table();

Ejemplo Dale un alias a una clase:

use Html\Table as T;
$table = new T();
