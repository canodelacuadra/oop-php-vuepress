# Clases en PHP

La definición básica de una clase comienza con la palabra reservada class, seguida de un nombre de clase, y continuando con un par de llaves que encierran las definiciones de las propiedades y métodos pertenecientes a dicha clase.

Por convención se suele utilizar UpperCamelCase (más conocido como PascalCase), cuando la primera letra de cada una de las palabras es mayúscula

Una clase puede tener sus propias constantes, variables (llamadas "propiedades"), y funciones (llamados "métodos").

<?php
class ClaseSencilla
{
    // Declaración de una propiedad
    public $var = 'un valor predeterminado';

    // Declaración de un método
    public function mostrarVar() {
        echo $this->var;
    }
}
?>

La pseudovariable $this está disponible cuando un método es invocado dentro del contexto de un objeto. $this es una referencia al objeto invocador (usualmente el objeto al cual el método pertenece

-> es el utilizado para referirse a un método o propiedad dentro de un objeto

Si el acceso es a una propiedad se omite el signo $ en el acceso a la misma

# Ejemplo de definción de la Clase Producto

Vamos a definir una clase con las siguientes propiedades y métodos:

  • clase : Producto
  • atributos : nombre, descripción , precio, cantidad
  • metodos : obtener nombre, cambiar nombre
<?php
class Producto {
  // Proiedades
    public $nombre;
    public $descripcion;
    public $precio;
    public $stock;
 

  // Métodos
  function set_nombre($_nombre) {
    $this->nombre = $_nombre;
       //La palabra this dentro de la clase hace referencia a sí misma
   // El signo flecha -> relaciona la clase con una propiedad o un método 
   // $_nombre es un parámetro que coge el nuevo valor, no tiene que ver en principio nada con el nombre de la propiedad por eso el guión bajo.
  }
  function get_nombre() {
    return $this->nombre;
 
  }
}

# Los objetos. Las instancias de la clase.

Para crear un objeto, éste se instancia de una clase. Se debe crear una variable y asignarla a la palabra reservada new seguido del nombre de la clase y doble paréntesis.

Las clases deberán ser definidas antes de su instanciación.

<?php
$instancia = new ClaseSencilla();

Podemos crear todos los objetos que deseemos a partir de una clase. Cada objeto tiene todas las propiedades y métodos definidos en la clase, pero las propiedades podrían diferentes valores .

En el siguiente ejemplo $platoDeGambas es una instancia de la clase Producto:

$platoDeGambas = new Producto();

Una vez instanciada podemos asignar valores a sus propiedades o ejecutar sus métodos. Un objeto hereda todas las propiedades y métodos de su clase.

$platoDeGambas->set_nombre('Plato de Gambas');
$platoDeGambas->stock =7;
$platoDeGambas->descripcion ='gambas rojas de Huelva';
echo $platoDeGambas->get_nombre();

La clase puede tener propiedades con valores por defecto pero el objeto en sí puede modificarlos...
